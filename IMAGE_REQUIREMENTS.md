# คู่มือรูปภาพบ้านพักล่องแพ

Phase 1 ยังไม่มีรูปจริง โฟลเดอร์รูปจึงมีเฉพาะ `.gitkeep` เพื่อรักษาโครงสร้างใน Git ห้ามดาวน์โหลดรูป Stock ใช้รูปจากเว็บไซต์อื่น หรือสร้างรูป AI ที่ทำให้แพและสถานที่ไม่ตรงกับของจริง

## หลักเกณฑ์รูป

- ใช้รูปจริงของบ้านพักล่องแพ เขื่อนศรีนครินทร์ ลูกค้า อาหาร และกิจกรรมจริงเท่านั้น
- ขอไฟล์ต้นฉบับที่คมชัด แล้วส่งออกเป็น WebP; สามารถเตรียม AVIF เพิ่มเพื่อใช้ภายหลัง
- ใช้ sRGB และควรบีบอัดให้ไฟล์ Hero ไม่เกินประมาณ 350 KB และรูป Card ไม่เกินประมาณ 250 KB โดยไม่เสียรายละเอียดสำคัญ
- ห้ามยืด บิด หรือ Crop จนโครงสร้างแพและกิจกรรมคลาดเคลื่อน
- ห้ามใส่ชื่อลูกค้า คะแนน หรือข้อความรีวิวลงบนภาพ หากยังไม่มีข้อมูลยืนยันและสิทธิ์เผยแพร่

## ขั้นตอนแทน Placeholder

1. ตรวจสอบว่ารูปเป็นสถานที่ แพ และกิจกรรมจริง และมีสิทธิ์ใช้งาน
2. Crop และส่งออกตามชื่อ ขนาด และสัดส่วนด้านบน โดยไม่เปลี่ยนโครงสร้างจริงของแพ
3. วางไฟล์ในโฟลเดอร์ที่ตรงกับประเภท ห้ามเปลี่ยนชื่อโฟลเดอร์
4. ลบ `.gitkeep` ในโฟลเดอร์นั้นได้เมื่อมีรูปจริงแล้ว
5. อ้างอิงรูปจากเว็บด้วย Path ที่ขึ้นต้นด้วย `/images/` เช่น `/images/rafts/a3/a3-cover.webp`
6. เมื่อใช้ `next/image` ให้ระบุ `width`, `height`, `sizes` และ Alt Text ภาษาไทยตามภาพจริง เพื่อป้องกัน Layout Shift
7. ใช้ `priority` เฉพาะรูป Hero แรก รูปนอกจอให้โหลดแบบ Lazy Loading
8. ตรวจทุก Breakpoint ว่าภาพไม่ยืด ไม่บิด ไม่ตัดบุคคลหรือส่วนสำคัญ และไม่มี Broken Image
